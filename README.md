# Yet another personal website

This website is using [Pelican](https://blog.getpelican.com/) and more specifically Kevin Deldycke's awesome theme [Plumage](https://github.com/kdeldycke/plumage)

## Quick Reference

### Install

```bash
$ poetry install
$ poetry shell
```

### Develop

```bash
$ make devserver
```

### Deploy

```bash
$ make github
```
